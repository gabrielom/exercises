## Nicer images (optional, Gemini "Nano Banana")

The SVG pictograms can be replaced per-exercise with AI-illustrated images
generated by Gemini 2.5 Flash Image. Get an API key at
[aistudio.google.com](https://aistudio.google.com) â†’ **Get API key**, then:

```
GEMINI_API_KEY=AIza... node tools/genimages.mjs push-up pigeon   # try a few
cp img/gen/push-up.png img/gen/_anchor.png                       # lock the style
GEMINI_API_KEY=AIza... node tools/genimages.mjs                  # the rest match it
```

Results land in `img/gen/<id>.png` and are picked up automatically
(`js/gen-manifest.js` tracks which exercises have generated art; everything
else keeps its SVG). Delete a PNG and rerun to redo it; `--force` regenerates.
Commit the PNGs and bump `VERSION` in `sw.js` to deploy.
