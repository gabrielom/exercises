# Exercises.

A minimalist exercise tracker PWA — **Gym / Stretching / Push-ups / Pull-ups** as a compact grid, every exercise with its own generated SVG pictogram and real working weights. Tap an exercise → fullscreen player: pick reps, hit **Done**, and it auto-advances to the next one (timers start and log themselves). Includes a guided **2-hour "Corpo" stretching routine** (front split · pancake · feet) built from the [Corpo playlist](https://youtube.com/playlist?list=PLrO-qJFePZuxuVqK3V3O8IFQKQl_fYzsu).

**Live:** https://gabrielom.github.io/exercises/ · **Plan:** [PLAN.html](https://gabrielom.github.io/exercises/PLAN.html)

- All data stays on your device (`localStorage`), with JSON export/import backup.
- Installable: open the live URL on your phone → share menu → **Add to Home Screen** → launches standalone and works offline.
- No build step, no framework, no dependencies. All 121 exercise figures are generated by `node tools/genfigs.mjs`.

## Enable GitHub Pages (one-time)

Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch → `main` / `(root)` → Save**.
The site appears at `https://gabrielom.github.io/exercises/` a minute later.

## Develop

Serve the folder with any static server and open it:

```
python3 -m http.server 8000
```

## Structure

```
index.html            app shell
css/style.css         theme + masonry layout
js/data.js            exercise catalog (real gym program + weights) + Corpo routine
js/app.js             grid, fullscreen auto-advancing player, history, export/import
js/routine.js         guided 60s-hold / 30s-rest routine player
img/                  121 generated SVG pictograms
tools/genfigs.mjs     figure generator (2D skeleton + equipment); poses in tools/poses.mjs
sw.js                 offline cache (bump VERSION to deploy updates)
manifest.webmanifest  PWA manifest
PLAN.html             project plan & full routine table
```
